# 🔐 Identity Security Framework for Supply Chain

A web-based security framework designed to manage identity, authentication, and access control within a supply chain system. This project focuses on securing user interactions and protecting sensitive data using modern authentication practices.

---

## 📌 Project Overview

This project is developed as a **Final Year BCA Project** using Python Flask. It provides a secure system where different users (such as admins and suppliers) can log in, access data based on their roles, and perform operations safely.

The goal is to enhance **identity security in supply chain environments** by implementing authentication, authorization, and session management mechanisms.

---

## 🚀 Features

- 🔑 User Registration & Login System  
- 🔐 Secure Authentication using Flask-Login  
- 👥 Role-Based Access Control (Admin / Supplier / User)  
- 🧠 Session Management  
- 🔒 Password Protection (hashed storage)  
- 📊 Dashboard Interface  
- 🗂️ Organized Template System using Jinja2  

---

## 🏗️ Tech Stack

- **Backend:** Python (Flask)  
- **Frontend:** HTML, CSS (Jinja Templates)  
- **Database:** SQLite  
- **Authentication:** Flask-Login
